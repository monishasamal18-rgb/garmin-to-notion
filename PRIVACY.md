Privacy Policy

This application is a personal automation tool that synchronizes activity data
from Garmin Connect to a Notion database.

Data Access
The application accesses Garmin activity data such as workouts, steps,
and health metrics through the Garmin API.

Data Usage
The data is used solely to create or update entries in the user's
personal Notion workspace.

Data Storage
No personal data is stored outside of Garmin Connect and Notion.
The integration does not share or sell user data.

Third Parties
This application interacts only with Garmin Connect and Notion APIs.

Contact
For questions regarding this integration, please contact the developer
through the GitHub repository.
